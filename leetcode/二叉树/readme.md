# 树
 - 层次计算规则：根节点为第1层，依次往下
 - 高度的计算规则：叶子节点高度为 1，依次往上
 - 度：一个节点开叉的个数

# 二叉树
 - 每个节点的度不超过2
 - 满二叉树：叶子节点都在最底层，除了叶子节点，每个节点都有左右两个子节点（每个节点的度都是 2）


# 遍历
 - 前序遍历：根节点 -> 左子树 -> 右子树
 - 中序遍历：左子树 -> 根节点 -> 右子树
 - 后序遍历：左子树 -> 右子树 -> 根节点



# DFS（深度优先）， BFS（广度优先）
 - 深度优先: 贯彻落实一条道走到底，不撞南墙不回头的思想。本质是栈结构
 - 广度优先: 地毯式层层推进，从起始顶点开始，依次往外遍历。本质是队列结构

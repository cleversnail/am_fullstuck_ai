# var, let, cont
1. var 声明的变量会存在声明提升（将变量的声明提升到当前作用域的顶部）let 不会
2. var 可以重复声明变量，let 不行
3. var 在全局声明的变量会默认添加在 woindow 对象上，let 不会
4. const 声明的变量值无法修改
